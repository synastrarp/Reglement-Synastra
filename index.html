Règlement Général — Synastra

Respect

## REGLEMENT OFFICIEL SYNASTRA ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 1. **RESPECT ET COMPORTEMENT** Le respect entre tous les joueurs est obligatoire. Les insultes, propos racistes, homophobes, sexistes ou discriminatoires sont strictement interdits. Le harcèlement ou les provocations répétées sont interdits. Tout conflit HRP doit être réglé avec le staff. Le spam, la publicité non autorisée et les comportements toxiques sont interdits. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 2. **ROLEPLAY GENERAL** Vous devez rester en personnage en permanence en jeu. Votre personnage doit agir de manière réaliste. Chaque action doit avoir une raison roleplay valable. Le fairplay est obligatoire dans toutes les scènes. Les scènes doivent être jouées jusqu'à leur conclusion. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 3. **METAGAMING** Le Metagaming est interdit. Il s'agit d'utiliser des informations obtenues hors du jeu (Discord, stream, amis, etc.) dans le roleplay. Exemples : Utiliser une information vue sur un stream. Connaître l'identité d'un joueur sans l'avoir apprise en RP. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 4. **POWERGAMING** Le Powergaming est interdit. Cela consiste à effectuer des actions irréalistes ou impossibles dans la vraie vie. Exemples : Transporter des objets impossibles. Ignorer les blessures graves. Forcer des actions RP à un autre joueur. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 5. **FREEHIT / FREEKILL / FREEARREST** Freekill : tuer un joueur sans raison RP. Freearrest : arrêter quelqu'un sans raison RP. Freehit : attaquer quelqu'un sans raison RP. Ces comportements sont interdits et sanctionnés. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 6. **VEHICLE DEATHMATCH (VDM)** Le VDM consiste à utiliser un véhicule pour tuer volontairement un joueur. Cette action est strictement interdite. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 7. **CAR RAM** Foncer volontairement dans un véhicule pour provoquer un accident est interdit. La conduite doit rester réaliste. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 8. **FEAR RP** Vous devez avoir peur pour la vie de votre personnage. Exemples : Si une arme est pointée sur vous, vous devez coopérer. Ignorer un danger évident est considéré comme un manque de FearRP. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 9. **NEW LIFE RULE (NLR)** Lorsque votre personnage meurt : Vous oubliez la scène qui a causé votre mort. Vous ne pouvez pas revenir sur les lieux pendant un certain temps. Vous ne pouvez pas vous venger pour votre mort. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 10. **PAIN RP** Vous devez jouer les blessures de votre personnage. Exemples : Si vous êtes blessé, vous devez agir comme tel. Courir ou se battre après une blessure grave est interdit. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 11. **COMBAT LOG** Quitter le jeu pour éviter une scène RP ou une sanction est interdit. Toute déconnexion pendant une scène peut entraîner une sanction. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 12. **MASS RP** Vous devez agir comme si la ville était remplie de citoyens. Exemples : Éviter les fusillades irréalistes en pleine ville. Respecter la présence fictive de civils. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 13. **ORGANISATIONS ET GANGS** Les organisations doivent respecter les règles du serveur. Les guerres entre organisations doivent avoir une raison RP. Les attaques gratuites sont interdites. Les scènes doivent rester réalistes. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 14. **BUGS ET EXPLOITS** L'utilisation de bugs, glitchs ou exploits est strictement interdite. Tout bug doit être signalé au staff. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 15. **ROLEPLAY STREAM** Le stream snipe est interdit. Utiliser un live pour obtenir des informations RP est sanctionné. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 16. **SANCTIONS** Le staff peut appliquer les sanctions suivantes : Warn Kick Ban temporaire Ban permanent Les décisions du staff sont finales. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 17. **REGLEMENT GENERAL** Le règlement peut être modifié à tout moment. En rejoignant le serveur, vous acceptez l'ensemble des règles. Le non-respect du règlement entraînera des sanctions. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Cheats

Les cheats et exploits sont interdits.

Roleplay

Le serveur Synastra est basé sur un roleplay sérieux.
